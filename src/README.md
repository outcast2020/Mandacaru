# Base App Structure

Estrutura inicial sugerida para a implementacao do frontend:

- `app/`: bootstrap, providers e rotas.
- `components/`: UI reutilizavel.
- `features/`: fluxos por tela.
- `game/`: motor, timer, score e selecao de palavras.
- `services/`: storage, ranking, analytics e share card.
- `styles/`: tokens, temas e estilos globais.
- `types/`: contratos TypeScript.
- `utils/`: helpers puros.

As pastas abaixo foram criadas como base para a proxima etapa de implementacao.
